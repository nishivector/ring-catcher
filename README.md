# Ring Catcher

Catch the falling rings! A Three.js game inspired by the Reddit trending topic about class rings.

## How to Play

- **Desktop:** Move mouse left/right to control the cup
- **Mobile:** Drag left/right on screen
- Catch rings to score points
- Chain catches for combo multipliers
- Difficulty increases over time
